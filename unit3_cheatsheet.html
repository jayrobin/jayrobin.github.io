Regular Expressions

A ruby cheatsheet

SymbolDescriptionExampleMatches
a Match any single character, such as a letter, number or punctuation, by typing it with no modifiers @ @
abc Match a sequence of characters by typing them with no modifiers hello hello
(a|b) Match one sequence OR another by separating them with a pipe character and enclosing them in paretheses (0|zero) 0, zero
[abc] Match a single character to one of any given characters by enclosing the group in square brackets h[aeiou]ll hall, hell, hill, holl, hull
[a-z] Match a single character from a range of characters by separating the start and end character in the sequence with a dash and enclosing in square brackets [0-9A-Z] 0, 1, 2, 3, ...A, B, C, D, E, ...
a? Match zero or one of the preceeding symbol dogs? dog, dogs
[abc]? class='left'Match zero or one of any of the preceeding group y[aeiou]?s ys, yas, yes, yis, yos, yus
[abc]? Match zero or one of the preceeding sequence my name (is )?bob my name is bob, my name bob
a+ Match one or more of a given symbol in sequence 5+ 5, 55, 555, 5555, ...
a* Match zero or more of a given symbol in sequence hel*o heo, helo, hello, helllo, ...
a{n} Match exactly n of the preceeding symbol in sequence hel{2}o hello
a{n,m} Match between n and m of the preceeding symbol in sequence what{2,4} whatt, whattt, whatttt
a{n,} Match at least n of the preceeding symbol in sequence hi{3,} hiii, hiiii, hiiiii, hiiiiii, ...
[^a-z] or [^abc] The carat negates the pattern inside the square brackets, which will therefore match anything except the given pattern for a single character h[^aeiou]llo hbllo, hcllo, h8llo, h!llo, ... (any character except a digit)
^abc The carat when used outside of square brackets indicates the start of the line, meaning that the regex will only match if the line starts with the pattern. ^123 123 (but not "hello 123")
abc$ class='left'Indicates the end of the line, meaning that the regex will only match if the line ends with the pattern. 123$ 123 (but not "123 hello")
\s Match any whitespace character, such as space, tab or newline (\S matches anything except a whitespace character) hi\s*there hithere, hi there, hi there, ...
\d Match any digit (\D matches anything except a digit) I have \d cats I have 0 cats, I 1 cats, I have 3 cats, ...
\w Match any word character, such as a lower/upper case letter, number or underscore (\W matches anything except a word character) h\wllo hello, hAllo, h8llo, h_llo, ...
