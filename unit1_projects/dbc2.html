What is Breadth-first search?

Breadth-first search is an algorithm for searching a tree structure. It differs from depth-first search in that it searches the entire breadth of the tree before moving onto the next level down.

The algorithm is as follows:

  1. Add the root node to the queue
  2. Get the next node in the queue (initially, the root node)
    • If the node is the one we're searching for, end the algorithm and return the node
    • Else, add any child nodes to the queue
  3. If the queue is empty, end the algorithm and return nil (search node not found)
  4. Continue from step 2.
