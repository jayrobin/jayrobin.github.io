Or: how I learned to stop worrying and love programming

Week Four - Technical blog

25 Jun 2014

Enumerating a collection

for index in 0..list.length   puts list[index] end for item in list   puts item end index = 0 while index < list.length   puts list[index]   index++ end item.each do |item|   puts item end

Looping a predetermined number of times

STUFF

for index in 0..10   puts index end index = 0 while index < 10   puts index   index++ end 10.times do |index|   puts index end

Looping between two given indices

for index in 10..20   puts index end index = 10 while index < 20   puts index   index++ end 10.upto(20) do |index|   puts index end

Looping while a condition is true

STUFF

while player.alive   player.move end player.move while player.alive

Looping until a condition is met

while !game_over   take_turn end until game_over   take_turn end take_turn until game_over
